ESEN

ESEN was the first group project in the UofU Coding Bootcamp. For this project, my group and I attempted to improve on the e-sports tracking page produced by ESPN. We worked with API's for SportRadar for the game and tournament tracking, and the API from TwitchTV for livestream status on the watched games. This project went unfinished due to the limited amount of time we had to work on it, but it has a lot of potential

My role on this project was the development of the UI. I developed the graphical theme of the site, as well as the integration of Materialize CSS for the buttons, modals, and carousel.
